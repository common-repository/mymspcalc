A small tool to calculate the price for MS Points

== Description ==

= Englisch =
myMSPCalc is a small tool to check your MS Points price. You simply add the amount of points you bought, the price and the amount of points you want to use to buy stuff. The calculator now tells you how much you payed for the content!

= Deutsch =
myMSPCalc ist ein kleines Tool um die Kosten für MS Points zu errechnen. Man trägt einfach die gekauften Points & Preis ein und natürlich die Menge Points die man verbrauchen möchte bzw. verbraucht hat. Der Rechner sagt einem dann, wieviel man effektiv für den Inhalt / das Spiel bezahlt hat.


== Installation ==

1. Upload folder `mymspcal` to the wordpress plugin directory
2. Activate the plugin in wordpress admin
3. Wrote a post or a side and put [mymspcalc] into it, to show the calculator
OR
4. Put &lt;?php showmyMSPCalc(); ?&gt; in your sidebar template to show the calculator
